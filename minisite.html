What's new in SQL Server BI?

Reporting Services
Learn more
SQL Server Reporting Services (SSRS) offers a single, on-premises solution for deploying and managing reports, letting you:
  • Create reports that are optimized for mobile devices and can be consumed on any device and form factor
  • Use the new web portal experience, which is built on HTML5 and optimized for modern web browsers
  • Design precisely paginated and formatted reports with modern design tools
Analysis Services
Learn more
SQL Server Analysis Services (SSAS) is an online analytical data engine that helps you transform complex data into a single model, making it easier to:
  • Get analytical data for business reports and client applications such as Excel, Reporting Services reports, and other third-party BI tools
  • Securely manage larger data models – both Multidimensional and Tabular
  • Transform complex schemas into a more business user-friendly environment to provide faster insights and increase user adoption
SQL Server 2016 enables you to connect Power BI to your on-premises Analysis Services models, giving you the option to:
  • Utilize the rich toolset of Power BI without having to move those datasets to the cloud
  • Access on-premises data any way you'd like with scheduled refreshes and live connections
  • Leverage existing investments while still benefitting from the power of the cloud
SQL Server 2016 enables you to integrate SQL Server business intelligence and reporting into SharePoint 2016, allowing you to:
  • Configure Scheduled Data Refresh for Excel workbooks that utilize Power Query connections
  • Access new Reporting Servces features – including Treemaps and Sunburst charts, the new HTML renderer, plugin-free printing, and export to PowerPoint
